## üöÄ Quick start

### Pre-requisites

- Node.js >= 12.18
- Docker >= 19.03.12

> Copy .env.example to .env and fill out with your own enviroment credentials

### Installation (Node)

```
$ cd gobarber-server
$ yarn
```

### Usage

Run all migrations

```sh
$ yarn typeorm migration:run
```

Start development server

```sh
$ yarn dev:server
```

### Installation (Docker)

Starting containers

```sh
$ docker-compose up
```

Stoping containers

```sh
$ docker-compose stop
```

## Tests

```sh
$ yarn test
```

## Docs

Start server
```
$ yarn dev:server
```

Access swagger docs by opening: [http://localhost:3333/api-docs](http://localhost:3333/api-docs)

## Lint

```sh
$ yarn lint
```
