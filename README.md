### Installation (Node)

```
$ cd gobarber-server
$ yarn
```

### Usage

Run all migrations

```sh
$ yarn typeorm migration:run
```

Start development server

```sh
$ yarn dev:server
```

### Installation (Docker)

Starting containers

```sh
$ docker-compose up
```

Stoping containers

```sh
$ docker-compose stop
```
